# p5.zine

This library was originally created by Iley Cao & Munus Shih in `code decolonized` for teahing a workshop on how to learn p5 through a zine-making process.

Our objective is to create a coded zine that focuses on digital identity. We will use P5.js to teach ‘function’ and generate a collaborative digital profile zine in the end. We created some customized functions for people to play with this zine more easily.

# How to use it

We have a empty template [here](https://editor.p5js.org/munusshih/sketches/RtFAXP3xN) on p5 web editor, you can just duplicate it and remix it!
